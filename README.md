# Cybersecurity Portfolio

A collection of small security-focused projects, built to learn and demonstrate core concepts.

## Projects

- [Password Strength Analyzer](https://github.com/arsheri07/cybersecurity-portfolio/tree/main/password-strength-analyzer) — client-side password checker with entropy-based scoring, dictionary/l33t-speak detection, and live breach checking via the Have I Been Pwned API.
- [Network Log Analyzer](https://github.com/arsheri07/network-log-analyzer) — Python tool that parses SSH auth logs and flags brute-force attacks (fast and slow/distributed) and off-hours logins on privileged accounts using sliding-window detection rules.
